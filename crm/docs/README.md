## 📋 문서 개요

이 저장소는 VIVE CRM 시스템의 기획, 설계, 개발, 테스트, 배포, 운영에 필요한 모든 문서를 포함하고 있습니다.

### 문서 구조

```
docs/

├── 02-시스템설계/         # 시스템 아키텍처 및 설계
├── 03-상세설계/           # 상세 설계 문서
├── 04-구현/               # 구현 가이드 및 구현 반영 문서
├── 05-테스트/             # 테스트 계획 및 결과
├── 06-배포/               # 배포 및 운영 문서
├── 07-유지보수/           # 유지보수 계획
├── 08-검토/               # 단계별 검토 산출물
```
