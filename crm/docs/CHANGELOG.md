## [1.0.0] - 2026-04-15

### Added (신규 기능)

#### 사용자 인증
- 이메일 기반 회원가입/로그인 기능
- JWT 기반 인증 시스템 (Access Token, Refresh Token)
- 비밀번호 재설정 기능

#### 고객 관리 (Contacts)
- 고객 정보 등록/조회/수정/삭제 (CRUD)
- 고객 프로필 360° 뷰
- 태그 기반 고객 분류
- CSV 일괄 임포트/익스포트
- 중복 고객 감지 및 병합

#### 영업 파이프라인 (Deals)
- 칸반 보드 방식 파이프라인 관리
- 5단계 영업 단계 (리드 → 기회 → 제안 → 협상 → 계약)
- 딜 금액 및 예상 마감일 관리
- 드래그 앤 드롭 단계 이동
- 파이프라인별 매출 예측

#### AI 기능
- AI 리드 스코어링 (0-100점, A/B/C/D 등급)
- 다음 행동 추천 (이메일/전화/미팅/휴식)
- 성공 확률 예측
- 매출 예측 리포트

#### 활동 관리 (Activities)
- 이메일/전화/미팅/메모 활동 기록
- 고객 타임라인 자동 생성
- 활동별 성과 분석

#### 작업/알림 (Tasks)
- 후속 조치 작업 등록
- 마감일 기반 알림
- 우선순위 설정 (높음/중간/낮음)
- 이메일/인앱 알림

#### 대시보드
- 핵심 KPI 요약 (총 고객, 신규 고객, 파이프라인 현황)
- 리드 스코어 분포 차트
- 오늘의 추천 행동 목록
- 미완료 작업 알림

#### 리포트
- 주간/월간 활동 리포트
- 영업 성과 분석
- 파이프라인 변화 추이
- 리드 스코어 변화 추적

#### 구독/과금
- Freemium 구조 (무료: 고객 100명 제한)
- Pro Starter 플랜 (₩29,000/월)
- Pro Business 플랜 (₩49,000/월)
- 연간 결제 할인 (17%)

### Security (보안)

- 모든 데이터 전송 TLS 1.2+ 암호화
- 비밀번호 bcrypt 해싱 저장
- JWT 기반 Stateless 인증
- SQL Injection 방지 (Prisma ORM)
- XSS 방지 (입력값 이스케이프)
- Rate Limiting 적용

### Technical (기술)

- Next.js 14+ (App Router) 프론트엔드
- Hono (Node.js) 백엔드 API
- PostgreSQL (Supabase/Neon) 데이터베이스
- Redis (Upstash) 세션/캐시
- Prisma ORM
- Tailwind CSS + shadcn/ui
- OpenAI API 연동

## 버전 관리 규칙

### Semantic Versioning

```
[MAJOR].[MINOR].[PATCH]
```

| 버전 | 설명 | 예시 |
|------|------|------|
| **MAJOR** | 하위 호환되지 않는 변경 | 1.0.0 → 2.0.0 |
| **MINOR** | 하위 호환되는 기능 추가 | 1.0.0 → 1.1.0 |
| **PATCH** | 하위 호환되는 버그 수정 | 1.0.0 → 1.0.1 |

### 카테고리 정의

| 카테고리 | 설명 |
|----------|------|
| **Added** | 새로운 기능 추가 |
| **Changed** | 기존 기능의 변경 |
| **Deprecated** | 곧 제거될 기능 |
| **Removed** | 제거된 기능 |
| **Fixed** | 버그 수정 |
| **Security** | 보안 관련 수정 |
