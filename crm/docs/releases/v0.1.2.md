# Release Notes: v0.1.2

Date: 2026-02-26

## Summary

This release introduces P1 priority features including Contact schema extensions, profile editing, CSV import, and AI recommendation APIs.

## Highlights

### New Features

- **Contact Schema Extensions**: Added `source`, `tags`, `memo`, `jobTitle`, `lastContactAt` fields
- **PATCH /api/v1/auth/me**: Profile editing endpoint (name, company)
- **POST /api/v1/contacts/import**: CSV bulk import with Content-Type: text/csv
- **GET /api/v1/deals/pipeline**: Kanban-style response with stages + deals array + summary
- **AI Recommendations**: 
  - GET `/api/v1/ai/recommendations` - Full recommendation list
  - GET `/api/v1/ai/recommendations/today` - Today's recommended actions
- **Tasks CRUD**: GET/PATCH/DELETE `/api/v1/tasks/:id` endpoints

### Business Logic

- AI lead score auto-calculation on contact create/update
- "Registration" activity auto-logged on contact creation
- Email/phone duplicate validation (BR-002-02)
- `lastContactAt` auto-update on activity registration
- A/B/C/D grade auto-calculation (SRS standard: A≥80, B 60-79, C 40-59, D<40)

### Schema Updates

- User.company field added
- LeadScore.grade field added
- Contact leadGrade uses A/B/C/D instead of HOT/WARM/COLD

## Migration
