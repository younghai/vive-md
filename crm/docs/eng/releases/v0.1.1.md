# Release Notes: v0.1.1

Date: 2026-02-26

## Summary

This release stabilizes API contract checks, updates auth error handling, and adds delivery automation for CI and Postman assets.

## Highlights

- OpenAPI and runtime routes are aligned (`33 documented`, `33 implemented`).
- `GET /api/v1/auth/me` now reliably returns `401` when auth context is missing.
- The contract audit script now resolves app imports correctly and excludes middleware-only `ALL` routes.
- Added Postman artifacts:
  - Collection: `docs/postman/crm-v1.postman_collection.json`
  - Environment: `docs/postman/crm-local.postman_environment.json`
- Added CI verification flow:
  - `npm run build`
  - `npm test`
  - `npm run contract:audit`
  - `npm run postman:generate`
